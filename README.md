# Rose Landing Page

Лендинг для креативного агентства Rose, созданный на основе дизайна из Figma.

## Технологии

- **React.js** - библиотека для создания пользовательских интерфейсов
- **Vite** - быстрый инструмент сборки
- **CSS3** - стилизация компонентов

## Структура проекта

```
rose-landing/
├── src/
│   ├── components/
│   │   ├── Header/       # Шапка сайта с навигацией
│   │   ├── Hero/         # Главный экран с розовым градиентом
│   │   ├── Services/     # Секция услуг (SMM, Дизайн, Фото и т.д.)
│   │   ├── Portfolio/    # Портфолио работ
│   │   ├── Clients/      # Клиенты и отзывы
│   │   ├── ContactForm/  # Форма обратной связи
│   │   └── Footer/       # Футер с контактами
│   ├── assets/           # Изображения и медиафайлы
│   ├── App.jsx           # Главный компонент приложения
│   └── main.jsx          # Точка входа
└── package.json
```

## Установка и запуск

### Установка зависимостей

```bash
npm install
```

### Запуск проекта в режиме разработки

```bash
npm run dev
```

Проект будет доступен по адресу: http://localhost:5173/

### Сборка для продакшена

```bash
npm run build
```

### Предпросмотр production-сборки

```bash
npm run preview
```

## Особенности

- Полностью адаптивный дизайн (responsive)
- Современный стек технологий
- Модульная структура компонентов
- Плавные переходы и анимации
- Sticky-навигация
- Форма обратной связи

## Секции лендинга

1. **Header** - навигационное меню с социальными ссылками
2. **Hero** - привлекательный главный экран с градиентом
3. **Services** - 6 основных услуг агентства
4. **Portfolio** - портфолио выполненных работ
5. **Clients** - отзывы и логотипы клиентов
6. **Contact Form** - форма для связи с агентством
7. **Footer** - контактная информация
