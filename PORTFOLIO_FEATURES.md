# Новые функции Портфолио

## Реализованные фичи:

### 1. Желтый фон
- Секция портфолио теперь имеет яркий желтый градиентный фон
- Градиент: от #ffe066 до #ffd700

### 2. Эффект выезжания карточек при скролле
- Карточки появляются с анимацией при прокрутке страницы
- Используется Intersection Observer API
- Карточки выезжают снизу вверх с плавным появлением
- Каждая карточка анимируется независимо

### 3. Модальное окно при клике на проект
**Функциональность:**
- Клик на любую карточку открывает модальное окно
- В модальном окне отображается:
  - Большое изображение проекта
  - Название проекта
  - Имя клиента
  - Год реализации
  - Теги услуг (Брендинг, SMM, Дизайн и т.д.)
  - Полное описание проекта

**Управление:**
- Закрытие по клику на фон (overlay)
- Закрытие по кнопке X в правом верхнем углу
- Кнопка X поворачивается при наведении

### 4. Кнопка "СМОТРЕТЬ ВСЕ"
**Функциональность:**
- Изначально показываются 6 проектов
- По клику на кнопку раскрываются все 9 проектов
- Кнопка меняет текст на "СКРЫТЬ"
- При скрытии происходит автоматическая прокрутка к секции портфолио

### 5. Эффекты при наведении (hover)
- Желтое свечение вокруг карточки
- Увеличение изображения (zoom)
- Подъем карточки вверх
- Выезжающее описание снизу

## Как редактировать контент:

### Изменить описание проекта:
Откройте `src/components/Portfolio/Portfolio.jsx` и отредактируйте массив `allPortfolioItems`:

```javascript
{
  title: 'НАЗВАНИЕ ПРОЕКТА',
  client: 'Имя клиента',
  image: 'URL изображения',
  description: 'Полное описание проекта...',
  year: '2024',
  services: ['Услуга 1', 'Услуга 2', 'Услуга 3']
}
```

### Добавить новый проект:
Добавьте новый объект в массив `allPortfolioItems` с теми же полями.

### Изменить количество показываемых проектов:
В строке 85 файла `Portfolio.jsx` измените число:
```javascript
const displayedItems = showAll ? allPortfolioItems : allPortfolioItems.slice(0, 6);
// Замените 6 на нужное количество
```

## Технические детали:

- **Intersection Observer** - отслеживает появление карточек в viewport
- **React Hooks**: useState, useEffect, useRef
- **CSS анимации**: fadeIn, slideUp, transform transitions
- **z-index модального окна**: 10000 (поверх всего контента)

## Адаптивность:

- **Desktop (>1024px)**: 3 колонки
- **Tablet (768px-1024px)**: 2 колонки
- **Mobile (<768px)**: 1 колонка
- Модальное окно адаптируется под размер экрана
